# Numina Mobile

React Native Expo app for emotional wellness platform.

## Quick Start

```bash
npm install
npx expo start
```

## Tech Stack

- React Native + Expo
- TypeScript
- React Navigation
- AsyncStorage

## Environment

Create `.env` file:
```env
EXPO_PUBLIC_API_URL=your_api_url
EXPO_PUBLIC_ENVIRONMENT=development
```

## Build

```bash
npx expo build:android
npx expo build:ios
```
